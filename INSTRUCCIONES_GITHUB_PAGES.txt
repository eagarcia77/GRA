GitHub Pages Deployment

Upload all files and folders from this package to the root of your GitHub Pages repository.
Required items:
- index.html
- viewer-ar.html
- v.html
- markers.html
- test-marker.html
- diagnostico-contenido.html
- css/
- js/
- assets/
- snippets/
- .nojekyll

Do not upload only index.html.
After uploading, wait 1–3 minutes and refresh the page.
