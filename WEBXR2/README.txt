UCAN AR Business Card - WebXR Surface Mode

Este paquete convierte la tarjeta del Dr. Eduardo García en una experiencia WebXR Surface Mode usando un modelo GLB.

Archivos principales:
- index.html: página principal.
- styles.css: estilos visuales.
- assets/eduardo_ucan_card.glb: modelo 3D en formato GLB.
- assets/tarjeta_ucan.png: textura usada en el modelo.
- assets/preview_ucan_ar_card.png: vista previa.

Cómo publicarlo:
1. Sube todo el contenido de esta carpeta a un hosting con HTTPS.
2. Puedes usar GitHub Pages, Netlify, Vercel o un servidor institucional.
3. Abre index.html desde el celular.
4. Presiona Ver en AR.
5. Apunta a una superficie y coloca la tarjeta.

Notas importantes:
- Para AR real, la página debe estar en HTTPS.
- En Android, Chrome suele usar WebXR o Scene Viewer.
- En iPhone, la visualización AR puede depender de Quick Look y de compatibilidad del navegador.
- El archivo GLB contiene la tarjeta como textura aplicada a una placa 3D.
